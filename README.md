# sentiment-analysis
Steps to understand sentiment polarization in user generated content (Text, Message, Blog, Comment, Review, and etc)

### Setup Alteryx and R
- Alteryx Web link: http://www.alteryx.com/download
- R for Mac download: https://cloud.r-project.org/
